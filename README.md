# Learnova — AI-Powered Learning Management System

A production-grade LMS combining RAG-powered AI study tools, gamification, real-time chat, and multi-tenant organization management. Built for schools and academies.

---

## Features

### AI & RAG
- **AI Mentor** — answers student questions grounded in lesson content via semantic retrieval (no hallucination)
- **Auto Quiz Generation** — generates scored quizzes from lesson attachments (PDF, DOCX, video)
- **Smart Flashcards** — auto-generated from lesson text, stored and surfaced per-lesson
- **Mind Maps** — hierarchical concept summaries generated by the AI engine

### Learning Experience
- **Gamification** — XP system, daily streaks, level progression, achievements, and live leaderboards
- **Lesson System** — video, PDF, and audio attachments with transcription; comments and progress tracking
- **Course Chat** — group and private messaging scoped to courses with soft-delete semantics
- **Marks & Analytics** — instructor dashboards with per-student performance, quiz attempts, and trends

### Platform
- **Multi-tenant** — organizations own courses → subjects → lessons; supports school and academy modes
- **Multi-role** — Admin, Organization, Instructor, Student, Parent — each with a dedicated UI
- **Bilingual** — full English/Arabic support across frontend and AI-generated content

## Architecture

```
┌─────────────────────────────────────────────────────────┐
│  React SPA (Vite)          Learnova_frontend/           │
└────────────────────┬────────────────────────────────────┘
                     │ REST API
┌────────────────────▼────────────────────────────────────┐
│  Express API (Node.js)     src/                         │
│  Controller → Service → Prisma → MariaDB                │
│                    │                                     │
│                    │ async trigger                       │
│  ┌─────────────────▼──────────────┐                     │
│  │  RAG Service (FastAPI)         │                      │
│  │  extract → chunk → embed       │                      │
│  │            │                   │                      │
│  │         Qdrant                 │                      │
│  └────────────────────────────────┘                     │
└─────────────────────────────────────────────────────────┘
```

**AI/RAG flow:**
1. Instructor uploads lesson attachments (PDF, DOCX, video, audio)
2. RAG service extracts text, transcribes media, and chunks content
3. Embeddings are stored in Qdrant with lesson/course metadata
4. Student queries → semantic retrieval → Groq LLM → grounded answer
5. Same pipeline powers quiz, flashcard, and mind map generation

**Gamification flow:**
- User actions (lesson complete, quiz pass) emit XP events persisted as immutable audit rows
- Streaks track consecutive daily activity
- Leaderboard snapshots are computed asynchronously for fast reads

## Getting Started

### Backend + Services (Docker Compose)

All backend services — API, MariaDB, Qdrant, and the RAG service — run via Docker Compose.

```bash
git clone <repo-url> && cd Learnova
cp .env.example .env          # fill in required values
docker compose up -d
```

### Frontend

```bash
cd Learnova_frontend
npm install
npm run dev
```

Frontend runs at `http://localhost:5173`.
